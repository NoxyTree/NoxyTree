## Cadence Starter + Approval Guard

**Your script changed. Your pipeline should notice.**

Two tested local Python tools for automation that must stop when reviewed inputs change. Approval Guard adds content-bound gates to an existing pipeline; Cadence Starter also tracks dependencies, catches stale work, blocks unsafe progress, and names the next safe action. Neither tool requires Claude, an AI account, or a subscription.
